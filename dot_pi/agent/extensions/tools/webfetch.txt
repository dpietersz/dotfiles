Fetch content from a URL using Crawl4AI. Returns clean markdown by default, can also return HTML or take a screenshot. Use this to read web pages, documentation, articles, or any web content.

Usage notes:
  - The URL must be a fully-formed valid URL starting with http:// or https://
  - Format options: "markdown" (default) extracts clean readable markdown, "html" returns preprocessed HTML, "screenshot" captures a page screenshot as an image
  - Use "markdown" for reading articles, docs, and general web content
  - Use "html" when you need the raw structure or when markdown conversion loses important layout
  - Use "screenshot" when you need to see the visual layout of a page
  - This tool is read-only and does not modify any files
  - Results may be truncated if the content is very large; full output is saved to a temp file
  - Timeout defaults to 30 seconds, max 120 seconds
  - Pages behind authentication or bot protection may fail
