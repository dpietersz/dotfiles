# Issue Title Rules

**Format**: `[Verb] [What] [Context]`

**Examples**:
- Story: `Enable users to reset password via email`
- Bug: `Fix double booking when time slots overlap`
- Task: `Refactor checkout API for Stripe v2`

**Rules**:
- Start with a verb: "Enable", "Fix", "Implement", "Design", "Refactor"
- Keep user stories in the description, not the title
- Do not prefix with type — rely solely on labels: `Improvement`, `Feature`, `Bug`, `Delegated`
